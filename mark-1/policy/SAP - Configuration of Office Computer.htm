2000-05-20-01

Configuration & Restoration of Office Computer

 

It shall be the policy of the Kempsville Volunteer Rescue Squad that the squad's primary business office computer shall have a known baseline hardware and software configuration that can be quickly, easily restored.

To this end, the following lists indicate the baseline hardware and software configuration of the squad's primary business office computer as of 14 August 2009.  This configuration has proven to be inexpensive and highly reliable.

Network

  • Single IP Internet service
  • Bridge
  • Wireless access point router with multi-port switch

Hardware

  • IBM PC compatible computer
  • High capacity hard disk drive
  • CD-ROM drive
  • Laser printer
  • Super VGA monitor
  • Keyboard
  • Mouse
  • Motorola Minitor III Pager Programmer cradle

Software

  • Microsoft Windows XP Pro, installed by:
    1. Booting from latest Microsoft Windows XP Pro installation CD
    2. Attaching to the network
    3. Running Windows Update
  • PDF reader
  • Motorola Radio Service Software for Minitor III pager
  • OpenOffice
  • Panda Cloud Antivirus
  • Revo Uninstaller
